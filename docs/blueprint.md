# **App Name**: MediaFlow

## Core Features:

- Project Dashboard: Display key project metrics, visual analytics, recent activity, and upcoming deadlines.
- Project List View: Allow users to view projects in a table or card format, with search, filtering, and sorting capabilities.
- Kanban Board: Enable drag-and-drop project management using a Kanban board with project stages as columns.
- Project Detail Page: Provide a detailed view of each project with sections for overview, timeline, tasks, notes, and files.
- Create/Edit Project Modal: Offer a modern modal UI for creating and editing projects with smart defaults and validation.
- Intelligent Progress Calculation: Automatically calculate project progress based on task completion using the LLM as a tool.
- Real-time Notifications: Provide real-time updates and deadline warnings via Firestore listeners.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to convey professionalism and trustworthiness.
- Background color: Light gray (#F5F7FA) for a clean, Apple-style UI.
- Accent color: muted green (#C1E1C1) for subtle highlights and calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern and neutral look.
- Use icons from Lucide, maintaining a consistent style across the app.
- Card-based layout with generous whitespace to enhance readability and visual appeal.
- Incorporate framer-motion style transitions and micro-interactions for a smooth user experience.
